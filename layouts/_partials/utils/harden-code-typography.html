{{- /* harden-code-typography: make the text inside an inline span immune to the Goldmark typographer, AND repair any typographer damage already present. Encodes the typographer's triggers (and the curly/dash glyphs it produces) as numeric character references: `--` and `–`/`–` both become `--`, smart quotes become `"`/`'`, etc. A browser renders `--` as `--`, but it is no longer a literal `--` pair, so neither this render nor any LATER one rewrites it to an en dash. Why one transform for every site: - Before a markdownify we control (callout/alert body): the `` holds a literal `--`; hardening stops the typographer from touching it. - After RenderString in reuse: a backtick span nested in a version / conditional-text block already rendered to `` and got re-typographed to `–`; hardening repairs it. - The killer case — `{{% reuse %}}` / `{{% conditional-text %}}` / `{{% version %}}` PERCENT forms: their output re-enters the page's own Goldmark, which re-typographs the `` a SECOND time AFTER the shortcode returns. A post-render *reversal* (en dash → `--`) would just be undone by that pass; only the entity form survives it. That is why we harden to `--` rather than reverse to `--`. Scope: attribute-less `` only — author/snippet inline code, never the Chroma `
` highlighted blocks (whose nested
       ``s also break the `[^<]` inner match). The ``/`` tags
       carry none of the trigger glyphs, so replacing across the matched span only
       ever rewrites inner text. Idempotent: re-hardening already-hardened content
       is a no-op (the entities contain no triggers).

       Input: a string of HTML (rendered, or raw markdown containing literal
       ``). Output (via return): the same string with inline-code
       typography hardened. Mirror copy lives in docs-theme-extras; keep in sync. */ -}}
{{- $s := . -}}
{{- range $c := findRE `(?s)[^<]*` $s -}}
  {{- $n := $c -}}
  {{- /* dashes: 3-char/em forms before 2-char/en forms so `---` keeps all three */ -}}
  {{- $n = replace $n "—" "---" -}}
  {{- $n = replace $n "—" "---" -}}
  {{- $n = replace $n "–" "--" -}}
  {{- $n = replace $n "–" "--" -}}
  {{- $n = replace $n "---" "---" -}}
  {{- $n = replace $n "--" "--" -}}
  {{- /* quotes */ -}}
  {{- $n = replace $n "“" """ -}}
  {{- $n = replace $n "”" """ -}}
  {{- $n = replace $n "“" """ -}}
  {{- $n = replace $n "”" """ -}}
  {{- $n = replace $n "\"" """ -}}
  {{- $n = replace $n "‘" "'" -}}
  {{- $n = replace $n "’" "'" -}}
  {{- $n = replace $n "‘" "'" -}}
  {{- $n = replace $n "’" "'" -}}
  {{- $n = replace $n "'" "'" -}}
  {{- if ne $n $c -}}
    {{- $s = replace $s $c $n 1 -}}
  {{- end -}}
{{- end -}}
{{- return $s -}}
