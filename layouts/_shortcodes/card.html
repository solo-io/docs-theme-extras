{{- /* Shortcode: card Renders a styled card link with an optional icon or image, title, and subtitle. Emits the same .section-card markup the parent-section auto-card grid uses, so manual {{< card >}} blocks look identical to auto-generated child cards. Parameters: - link: (optional) A direct URL for the card link (used as-is). - path: (optional) A section-relative path resolved to the current docs section (e.g., "/tutorials/basic/"). - title: The card title (aliased as "header"). - subtitle: (optional) Short description below the title (aliased as "description"); supports Markdown. - icon: (optional) Name of an icon to render via utils/icon.html. - image: (optional) Image URL to display at the top of the card. - alt: (optional) Alt text for the image; defaults to the title. Title and subtitle are emitted as with display: block via CSS — NOT

— so the card stays valid HTML when it's wrapped by markdownify (e.g., from a percent-form {{%% %%}} shortcode). A nested

inside the outer

Goldmark adds would trigger the browser's auto-close-the-outer-

behavior, closing the too early and orphaning the description. Example: {{< card title="Get started" subtitle="Learn the basics" path="/getting-started/" icon="rocket" >}} */ -}} {{- $link := .Get "link" -}} {{- $path := .Get "path" -}} {{- if $path }} {{- $pathTrimmed := strings.TrimPrefix "/" $path -}} {{- /* Resolve the docs section prefix via the shared page-context partial so this shortcode picks up every section the partial knows about (kubernetes, standalone, envoy, agentgateway, ...) without hardcoding a regex here. Falls back to FirstSection.RelPermalink for pages outside the /docs/

// shape. */ -}} {{- $ctx := partial "utils/page-context.html" . -}} {{- $section := $ctx.prefix -}} {{- if not $section }}{{ $section = strings.TrimSuffix "/" .Page.FirstSection.RelPermalink }}{{ end -}} {{- $section = replaceRE `^https?://[^/]+` "" $section -}} {{- $section = replaceRE `^/localhost:[^/]+` "" $section -}} {{- if not (hasPrefix $section "/") }}{{ $section = printf "/%s" $section }}{{ end -}} {{- $link = printf "%s/%s" $section $pathTrimmed -}} {{- if and (not (strings.HasSuffix $link "/")) (not (in $link "#")) }}{{ $link = printf "%s/" $link }}{{ end -}} {{- range (seq 10) }}{{ $link = replace $link "//" "/" }}{{ end -}} {{- end -}} {{- $title := .Get "title" | default (.Get "header") -}} {{- $subtitle := .Get "subtitle" | default (.Get "description") -}} {{- $icon := .Get "icon" -}} {{- $image := .Get "image" -}} {{- $alt := .Get "alt" | default $title -}} {{- if $image -}} {{- end -}} {{- if $icon -}} {{ partial "utils/icon.html" (dict "name" $icon "attributes" "class=\"section-card-icon\"") }} {{- end -}} {{- if $title }}{{ $title }}{{ end -}} {{- if $subtitle }}{{ $subtitle | markdownify }}{{ end -}}
