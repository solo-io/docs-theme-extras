Accepts a body from either the inner content (block form) or a self-closing text="…" attribute. Two modes: - hugo.Environment "translation": emit the callout shortcode itself as open/close placeholders (restored verbatim by copy-markdown.html AFTER transform.HTMLToMarkdown) with the body flowing through between them, so the translation-export snapshot keeps the shortcode while the prose stays translatable. The store is re-read between the open and close registrations so a placeholder a nested shortcode registers while the body renders keeps its order. No-op for normal/preview/prod builds. - otherwise: render the solo-alert div on ONE logical line (body newlines → ) so a callout nested in a list item doesn't trip Goldmark's content-continuation column rule and break the list. a11y: role="note" marks the box as ancillary content (the correct ARIA role for a static admonition — NOT role="alert", which is an assertive live region meant for dynamically-injected messages). The icon is aria-hidden; it's decorative and the body text carries the meaning. */ -}} {{- $type := .Get "type" | default (.Get "context") -}} {{- $body := .Get "text" | default .Inner -}} {{- /* Dedent the body so CommonMark's "4 spaces = code block" rule doesn't fire when the shortcode is nested in a deeply-indented list item.
