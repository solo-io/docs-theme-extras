{{- /* Shortcode: gloss Renders an inline glossary term with a hover/focus tooltip showing the definition and an optional "Learn more" link. Looks up the term in the consumer's `data/glossary.yaml`. Parameters: - 0 (positional): The glossary key (e.g. "MCP"). - (inner content): Optional display text overriding the key. Tooltip positioning + behavior is driven by extras' glossary.js (position:fixed so tooltips escape overflow:auto ancestors like .table-wrapper). Styling lives in extras' glossary.css. Both are loaded automatically by themeExtras/head-end.html. Example: {{< gloss "MCP" >}}custom display text{{< /gloss >}} */ -}} {{- $key := .Get 0 -}} {{- $entry := index site.Data.glossary $key -}} {{- if $entry -}} {{ .Inner | default $key }}{{ $key }}{{ $entry.short }}{{- if $entry.link }}Learn more{{- end }} {{- else -}} {{ .Inner | default $key }} {{- end -}}
